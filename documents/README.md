# 数字分身 — 文档索引 / Digital Twin — Documentation Index

## 项目简介 / Project Overview

**数字分身 (Digital Twin)** 是一个基于本地开源大语言模型的 AI 克隆项目，支持多平台交互。

**Digital Twin** is a local OSS LLM-based AI clone project with multi-platform support.
