# 设计决策记录 / Architecture Decision Records (ADR)

## 概述 / Overview

本文档记录项目的关键架构决策，包含上下文、决策理由和影响。

This document records key architectural decisions, including context, rationale, and consequences.

---

## ADR 模板 / ADR Template

每条 ADR 使用以下格式 / Each ADR uses the following format:

```markdown
# ADR-{NNN}: {标题 / Title}

## 状态 / Status

{Proposed / Accepted / Deprecated / Superseded}

## 日期 / Date

YYYY-MM-DD

## 上下文 / Context

描述需要做出决策的问题和背景。
Describe the problem and background requiring a decision.

## 决策 / Decision

描述做出的决策及其理由。
Describe the decision made and its rationale.

## 后果 / Consequences

### 正面 / Positive
- [待定 / TBD]

### 负面 / Negative
- [待定 / TBD]

### 风险 / Risks
- [待定 / TBD]
```
